# CareerLens AI

Upload or paste a resume/profile and let Gemini generate a personalized career advisor report with role recommendations, skill gaps, hiring probability, salary guidance, ATS fixes, interview prep, and a 7-day action plan.

## Run locally

```powershell
$env:GEMINI_API_KEY="your_api_key_here"
npm start
```

Open `http://localhost:3000`.

## Test

```powershell
npm test
```

## Notes

- The Gemini API key is read only on the backend.
- The MVP prioritizes pasted or plain-text resume/profile input for reliable hackathon demos.
- Salary and hiring probability values are advisory estimates generated by Gemini, not guaranteed employment or compensation outcomes.
